TASK 009 — 철퇴의 브루노 보스와 자동 재도전 루프 구현
작업 목적
첫 지역의 최종 보스인 `철퇴의 브루노`를 구현한다.
이번 작업에서 검증할 흐름은 다음과 같다.

```text
일반 적 사냥
→ 오우거 징수꾼 처치
→ 철퇴의 브루노 등장
→ 기본 공격·강공격·방어 태세
→ 특성에 따라 다른 방식으로 공략
→ 패배 원인 안내
→ 일반 사냥으로 자동 복귀
→ 성장 후 다시 보스 도전
→ 보스 처치
→ 첫 지역 클리어

```

핵심 질문은 다음과 같다.
엘리트보다 한 단계 강한 보스가 등장했을 때 성장하고 다시 도전하고 싶은 마음이 생기는가.
이번 작업에서는 보스 전투와 재도전까지만 구현한다.
빚 상환과 명성 상승은 구현하지 않는다.

전체 진행 순서
보스 등장 전 순서
현재 한 순환은 다음과 같다.

```text
늑대
→ 늑대
→ 늑대
→ 고블린 무리 3마리
→ 방패병
→ 오우거 징수꾼

```

오우거 징수꾼을 처치하면 철퇴의 브루노가 등장한다.
전체 흐름은 다음과 같다.

```text
일반 적 순환
→ 오우거
→ 철퇴의 브루노

```

보스에게 패배하면 첫 늑대로 복귀한다.
그 뒤 일반 적과 오우거를 다시 처치하면 보스에게 재도전한다.
보스 상태
다음 상태를 추가한다.

```text
보스 처치 여부.
보스 도전 예정 여부.
보스 전투 중 여부.
보스 전투 제한 시간.
보스 방어 태세 여부.
보스 자세 붕괴 여부.
자세 붕괴용 연속 타격 수.
연속 타격 유효 시간.

```

권장 상태 이름의 예시는 다음과 같다.

```gdscript
var boss_defeated := false
var boss_due := false
var boss_fight_active := false
var boss_time_left := 45.0

```

정확한 변수명은 현재 코드 스타일에 맞게 조정할 수 있다.
저장 기능은 아직 없으므로 게임을 새로 실행하면 보스 미처치 상태로 시작한다.
철퇴의 브루노 프로필
기본 수치
다음 프로필을 적 데이터에 추가한다.

```text
이름: 철퇴의 브루노
종류: 보스
최대 체력: 850
공격력: 15
기본 방어력: 6
경험치 보상: 100
골드 보상: 200
기본 공격 간격: 2.0초
접근 속도: 80~100px/s
보스 전투 제한 시간: 45초

```

접근 속도는 현재 화면에서 지나치게 오래 기다리지 않는 범위로 조정할 수 있다.
프로필 수치와 보상은 한 곳에서 관리한다.
별도의 데이터베이스나 범용 보스 편집기는 만들지 않는다.
보스 등장
등장 조건
다음 조건을 모두 만족하면 보스를 등장시킨다.

```text
오우거 처치.
보스 미처치.
현재 고블린 대기열 없음.
현재 적 노드 정리 완료.

```

오우거를 처치하면 `boss_due` 상태로 전환한다.
다음 적 생성 시 일반 적 대신 브루노를 생성한다.
등장 연출
보스가 등장하면 다음 문구를 표시한다.

```text
보스 등장!
철퇴의 브루노

```

표시 시간은 약 1초에서 1.5초로 한다.
전투를 장시간 멈추지 않는다.
보스 전투가 시작되면 제한 시간 45초를 표시한다.
보스 외형
완성 아트는 사용하지 않는다.
임시 도형으로 일반 적과 엘리트보다 강한 존재처럼 보여야 한다.
최소한 다음을 적용한다.

* 오우거보다 크거나 비슷하지만 더 넓은 실루엣.
* 짙은 적색 또는 자주색 계열.
* 큰 철퇴 도형.
* 일반 적보다 넓은 보스 체력 바.
* `[보스] 철퇴의 브루노` 이름표.
* 전투 화면 상단 또는 중앙 상단의 보스 전용 정보 영역.
철퇴는 손잡이와 큰 철구를 단순 도형으로 표현해도 된다.
방패병이나 오우거의 색상만 바꾼 모습으로 만들지 않는다.
보스 전용 UI
보스 전투 중 다음 정보를 표시한다.

```text
[보스] 철퇴의 브루노
HP 850 / 850
남은 시간 45초
현재 상태

```

현재 상태는 다음 중 하나다.

```text
일반.
강공격 준비.
방어 태세.
자세 붕괴.

```

보스 전투가 끝나면 보스 전용 UI를 숨긴다.
일반 적 전투 중에는 표시하지 않는다.
모바일 세로 화면에서 기존 상태 표시·경험치 바·특성 UI와 겹치지 않게 한다.
보스 공격 패턴
브루노는 다음 공격 리듬을 반복한다.

```text
1번째 공격: 기본 공격.
2번째 공격: 기본 공격.
3번째 공격: 강한 내려찍기.

```

즉 다음 패턴이다.

```text
기본
→ 기본
→ 강공격
→ 기본
→ 기본
→ 강공격

```

오우거와 같은 세 번째 공격 리듬을 사용하지만, 브루노는 방어 태세와 자세 붕괴를 추가로 가진다.
보스가 사망하거나 전투가 종료되면 공격 카운터를 초기화한다.
보스 등장 시 용병의 반격 카운터도 0으로 초기화한다.
따라서 반격 특성은 브루노의 세 번째 강공격에 맞춰 발동해야 한다.
기본 공격
브루노의 기본 공격은 공격력 15를 사용한다.

```text
실제 피해 = 15 - 용병 방어력
최소 피해 = 1

```

용병 기본 방어력 2라면 다음과 같다.

```text
15 - 2 = 13 피해

```

기존 피해 숫자와 피격 표현을 사용한다.
강한 내려찍기
공격 수치
브루노의 세 번째 공격은 강공격이다.

```text
기술 공격력 = 브루노 공격력 × 2.3
실제 피해 = 기술 공격력 - 용병 방어력
최소 피해 = 1

```

용병 기본 방어력 2라면 다음과 같다.

```text
round(15 × 2.3) - 2
= 33 피해

```

준비 동작
강공격 전 약 1.2초의 준비 동작을 표시한다.
준비 중에는 다음이 보여야 한다.

* 철퇴를 위로 크게 들어 올림.
* 몸체가 붉거나 주황색으로 강조됨.
* `강한 내려찍기 준비!` 문구.
* 공격 전까지 피해 없음.
준비 중에도 용병은 계속 공격한다.
준비 중 브루노가 죽거나 자세가 붕괴되면 강공격을 취소한다.
죽은 보스나 죽은 용병이 공격을 이어가면 안 된다.
반격과 보스 강공격
반격 특성은 적의 세 번째 공격마다 발동한다.
브루노의 세 번째 공격은 강공격이므로 다음 흐름이 발생한다.

```text
브루노 강공격.
→ 반격 발동.
→ 받는 피해 40% 감소.
→ 용병이 생존하면 즉시 반격.

```

기본 강공격 피해가 33이라면 다음과 같다.

```text
round(33 × 0.6)
= 20 피해

```

반격 피해는 기존 규칙을 사용한다.

```text
반격 피해
= round(용병 공격력 × 1.2) - 브루노의 현재 방어력
최소 피해 = 1

```

문구 예시:

```text
강공격 방어 -20
반격 -9

```

정확한 반격 피해는 현재 공격력과 보스 상태에 따라 달라진다.
방어 태세
발동 주기
보스 전투 시작 후 10초마다 방어 태세를 사용한다.

```text
방어 태세 주기: 10초.
지속 시간: 3초.

```

강공격 준비 중이거나 자세 붕괴 상태이면 즉시 시작하지 않는다.
해당 상태가 끝난 뒤 시작한다.
효과
방어 태세 중 방어력을 50% 증가시킨다.

```text
기본 방어력 6.
방어 태세 방어력 round(6 × 1.5) = 9.

```

방어 태세 중에는 몸체 또는 철퇴에 방어 효과를 표시한다.
문구:

```text
방어 태세!

```

보스는 방어 태세 중에도 기본 공격을 계속할 수 있다.
단, 자세 붕괴가 발생하면 방어 태세를 즉시 해제한다.
연격과 자세 붕괴
연격이 보스전에서 의미를 가지도록 자세 붕괴 규칙을 구현한다.
연속 타격 조건
짧은 시간 동안 브루노가 8회의 피해를 받으면 자세가 붕괴된다.

```text
필요 타격 수: 8회.
유효 시간: 첫 타격부터 4초.
자세 붕괴 지속 시간: 3초.

```

4초 안에 8타를 채우지 못하면 타격 수를 0으로 초기화한다.
다음 공격은 각각 1타로 계산한다.

* 일반 공격.
* 연격 추가 베기.
* 강타.
* 반격.
피해량의 크기와 관계없이 실제 피해가 적용된 한 번을 1타로 센다.
연격의 추가 베기도 별도 1타이므로 연격 특성이 자세 붕괴를 빠르게 만들 수 있어야 한다.
자세 붕괴 효과
자세가 붕괴되면 다음을 적용한다.

```text
3초 동안 공격 중단.
강공격 준비 중이면 취소.
방어 태세 중이면 즉시 해제.
방어력을 기본 방어력보다 40% 낮춤.

```

브루노의 자세 붕괴 방어력은 다음과 같다.

```text
round(6 × 0.6)
= 4.

```

자세 붕괴 중에는 다음 문구를 표시한다.

```text
자세 붕괴!

```

보스가 흔들리거나 잠시 주저앉는 단순 표현을 추가한다.
자세 붕괴가 끝나면 다음을 처리한다.

* 방어력 기본값으로 복귀.
* 공격 타이머 초기화.
* 연속 타격 수 초기화.
* 방어 태세 타이머는 다시 진행.
자세 붕괴 중 다시 8타를 채워 지속 시간을 연장하지 않는다.
보스의 현재 방어력
보스에게 피해를 줄 때 현재 상태를 기준으로 방어력을 계산한다.
우선순위는 다음과 같다.

```text
1. 자세 붕괴 중: 방어력 4.
2. 방어 태세 중: 방어력 9.
3. 일반 상태: 방어력 6.

```

자세 붕괴는 방어 태세보다 우선한다.
모든 용병 공격은 같은 현재 방어력 값을 사용한다.

* 일반 공격.
* 강타.
* 연격.
* 반격.
코드 여러 곳에서 각기 다른 방식으로 방어력을 계산하지 않는다.
작은 보조 함수 하나로 현재 방어력을 반환해도 된다.
강타의 보스 추가 피해
`UNKNOWN.md`에 정의된 강타 1단계의 보스 추가 피해 10%를 이번 작업에서 처음 적용한다.
일반 적과 엘리트에게는 기존 강타 공식을 유지한다.
브루노에게만 다음 공식을 사용한다.

```text
기술 공격력
= round(용병 공격력 × 2.2 × 1.1)

최종 피해
= 기술 공격력 - 브루노의 현재 방어력

최소 피해
= 1

```

예를 들어 용병 공격력 10, 보스 일반 방어력 6이면 다음과 같다.

```text
round(10 × 2.2 × 1.1) - 6
= 18 피해

```

보스 추가 피해가 두 번 적용되지 않게 한다.
연격의 보스 역할
연격 수치와 확률은 기존 규칙을 유지한다.

```text
일반 공격 시 30% 확률.
추가 베기 60% 피해.
강타 공격에서는 발동하지 않음.
추가 베기에서 재발동하지 않음.

```

브루노 전용 추가 피해는 주지 않는다.
연격의 역할은 다음이다.

```text
짧은 시간에 타격 수를 쌓아 자세 붕괴를 일으킨다.

```

연격 없이도 자세 붕괴가 불가능하지는 않지만, 연격이 가장 빠르게 조건을 달성해야 한다.
보스 제한 시간
보스 전투에는 45초 제한 시간을 둔다.
시간은 브루노와 실제 전투가 시작된 순간부터 감소한다.
등장 연출과 접근 중에는 감소시키지 않는다.
다음 상황에서는 시간을 감소시키지 않는다.

* 게임이 일시 정지된 경우.
* 특성 패널을 열어 게임 자체를 멈추도록 구현한 경우.
기존 특성 패널이 전투를 멈추지 않는다면 시간도 계속 흐르게 한다.
남은 시간이 0이 되었는데 보스가 살아 있으면 `화력 부족`으로 처리한다.
패배 원인
생존 부족
용병의 체력이 0이 되면 다음 문구를 표시한다.

```text
보스 도전 실패.
생존 부족.

체력·방어력을 강화하거나
반격 특성을 사용해 보세요.

```

화력 부족
45초 제한 시간이 끝나면 다음 문구를 표시한다.

```text
보스 도전 실패.
화력 부족.

공격력을 강화하거나
강타·연격 특성을 사용해 보세요.

```

정답을 강제하지 않고 해결 방향만 알려준다.
팝업이나 문구는 약 1.5초에서 2초 동안 보여준다.
버튼 입력을 요구하지 않고 자동으로 일반 사냥으로 복귀한다.
패배 후 자동 복귀
보스에게 패배하면 다음을 처리한다.

```text
브루노 제거.
보스 UI 제거.
보스 타이머 제거.
방어 태세·자세 붕괴 상태 초기화.
용병 부활 또는 체력 완전 회복.
첫 늑대로 복귀.
일반 적과 오우거를 다시 진행.
오우거 처치 후 브루노 재도전.

```

다음 성장 상태는 유지한다.

* 레벨.
* 경험치.
* 골드.
* 공격력·체력·방어력 강화.
* 특성 투자 상태.
* 보유 특성 포인트.
브루노의 체력은 재도전할 때 850으로 초기화한다.
패배 중에는 골드와 경험치를 지급하지 않는다.
보스 승리
브루노의 체력이 0이 되면 다음 보상을 정확히 한 번 지급한다.

```text
골드 +200.
경험치 +100.

```

반격으로 마지막 피해를 주어도 한 번만 지급한다.
경험치 보상으로 레벨업 조건을 넘으면 기존 레벨업 처리를 즉시 실행한다.
승리 문구를 표시한다.

```text
보스 처치!
철퇴의 브루노 격파.

첫 지역 클리어!

```

이번 작업에서는 빚 상환과 명성 상승을 표시하지 않는다.
보스 처치 후 진행
보스를 처치하면 `boss_defeated = true`로 기록한다.
이번 실행 세션에서는 브루노가 다시 등장하지 않는다.
약 2초의 승리 표시 후 첫 늑대로 돌아가 일반 사냥을 계속한다.
보스 처치 이후의 흐름은 다음과 같다.

```text
첫 늑대로 복귀.
→ 일반 적과 오우거 순환 계속.
→ 브루노는 다시 등장하지 않음.

```

게임을 완전히 새로 실행하면 저장 기능이 없으므로 브루노 미처치 상태로 초기화된다.
저장 기능은 다음 작업 이후 별도로 구현한다.
보스 진행 표시
일반 전투 중 보스까지 남은 진행 상황을 간단히 표시한다.
예시:

```text
보스까지 6 / 8

```

현재 순환은 다음 8단계로 계산할 수 있다.

```text
늑대 3.
고블린 3.
방패병 1.
오우거 1.

```

고블린은 개별 처치 기준으로 계산한다.
오우거 처치 후에는 다음처럼 표시한다.

```text
보스 등장 예정.

```

보스 전투 중에는 다음처럼 표시한다.

```text
보스 전투.

```

보스 처치 후에는 다음처럼 표시한다.

```text
첫 지역 완료.

```

기존 상단 UI가 지나치게 복잡해지면 짧은 한 줄로 표시한다.
전투 상태 정리
보스에게 필요한 최소 필드만 현재 적 Dictionary에 추가한다.
예시:

```text
is_boss.
attack_count.
is_winding_up.
windup_timer.
defense_stance.
defense_timer.
defense_cooldown.
staggered.
stagger_timer.
posture_hits.
posture_window_timer.

```

변수명은 현재 코드 스타일에 맞게 조정할 수 있다.
범용 보스 AI, 행동 트리, 복잡한 상태머신은 만들지 않는다.
현재 `Battle.gd`에서 관리 가능한 최소 구조를 유지한다.
오우거와 보스의 구분
브루노가 체력만 늘어난 오우거처럼 느껴지지 않아야 한다.
차이는 다음과 같다.

```text
오우거:
기본·기본·강공격.
반격의 역할을 확인하는 엘리트.

브루노:
기본·기본·강공격.
방어 태세.
연격에 의한 자세 붕괴.
45초 제한.
패배 원인.
성장 후 재도전.

```

오우거의 수치나 패턴을 이번 작업에서 변경하지 않는다.
자동 검증
기존 검증 유지
TASK 001~008의 자동 검증을 모두 유지한다.
다음 항목을 삭제하거나 완화하지 않는다.

* 자동전투.
* 성장과 강화.
* 레벨업.

공격 패턴 검증
브루노의 첫 6회 공격 유형을 확인한다.

```text
기본.
기본.
강공격.
기본.
기본.
강공격.

```

강공격 준비가 완료되기 전에 피해가 발생하면 실패다.
준비 중 보스를 죽이거나 자세 붕괴시키면 강공격이 취소되어야 한다.
공격 피해 검증
용병 방어력 2를 기준으로 한다.

```text
기본 공격:
15 - 2 = 13.

강공격:
round(15 × 2.3) - 2 = 33.

```

반격 발동 시 강공격 피해는 다음과 같다.

```text
round(33 × 0.6) = 20.

```

같은 강공격에서 반격이 두 번 발생하면 안 된다.
방어 태세 검증
다음을 확인한다.

```text
기본 방어력 6.
방어 태세 방어력 9.
10초마다 발동.
3초 동안 유지.
종료 후 방어력 6.

```

강공격 준비 중에는 방어 태세가 겹쳐 시작되지 않아야 한다.
자세 붕괴가 발생하면 방어 태세가 즉시 종료되어야 한다.
자세 붕괴 검증
검증용으로 4초 안에 8회의 피해를 강제로 적용한다.
다음을 확인한다.

```text
8번째 피해에서 자세 붕괴.
3초 동안 유지.
방어력 4.
공격 중단.
강공격 준비 취소.
방어 태세 취소.
종료 후 방어력 6.
타격 수 초기화.

```

4초 안에 7회만 공격하고 시간이 끝나면 타격 수가 0으로 초기화되어야 한다.
자세 붕괴 중에는 다시 자세 붕괴가 연장되지 않아야 한다.
강타 보스 추가 피해 검증
용병 공격력 10, 브루노 일반 방어력 6을 기준으로 한다.

```text
round(10 × 2.2 × 1.1) - 6
= 18.

```

일반 적과 오우거에게는 보스 추가 피해 10%가 적용되지 않아야 한다.
