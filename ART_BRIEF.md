# 균열기사 — 아트 외주 발주서 (Art Order)

세로형 모바일 방치형 RPG. **이 문서 하나로 전체 발주가 되도록** 현재 구현된 모든 아트 자산을 정리했다.
아티스트에게 이 파일을 그대로 전달하면 된다. (개발자 = 발주자, 아트 = 전량 외주.)

---

## 0. 목표 스타일 (확정)

**밝은 SD 상용 방치/수집 RPG 톤** (레퍼런스: 버섯커 키우기·세븐나이츠류 광고의 밝고 컬러풀한 SD).
- **밝고 채도 높은 컬러.** 어둡거나 머디한 톤 금지.
- **SD/치비 캐릭터** — 2~3등신, 큰 머리·또렷한 눈, 과장된 무기·이펙트, 귀엽지만 강렬.
- **깔끔한 애니풍 셀 셰이딩** + 또렷한 외곽선. 가독성 최우선.
- **크고 화려한 보스** + 폭발적 이펙트.

### 절대 규칙 (1차 실패 원인 — 반드시 지킬 것)
1. 배경은 밝되 **캐릭터보다 채도·디테일 낮춰 후퇴**. 캐릭터/보스는 배경과 **명도·실루엣 대비**로 또렷이 분리.
2. **주인공 ≠ 보스** — 주인공은 밝고 영웅적인 인간형, 보스는 거대·다른 색·다른 생물.
3. **보스 그림 = 이름에 맞는 생물** (거미=거미, 골렘=골렘, 용=용 — §4 표대로).
4. "전체 한 색(특히 보라) 덩어리" 금지. 지역별 색 변주.

## 1. 공통 기술 규격 (전 자산)

- **포맷**: PNG, **RGBA 투명배경**(체커·흰배경 금지). 배경만 불투명 가능.
- **해상도**: 고해상 일러스트로 작업 후 내보내기. 주인공·일반 적 **~512px 높이**, 보스 **~768~1024px**, 보스 초상화 **512×512**, 배경 **1080×1920(@2x) + 540×960(@1x)**, 아이콘 **256×256**.
- **뷰/방향**: 횡전투. **주인공은 오른쪽, 적·보스는 왼쪽을 바라봄**(또는 정면 3/4 통일). **피벗 = 발 바닥 중앙**(접지용). 캔버스 가장자리 2~3px 여백.
- **애니메이션**: 가로 스프라이트 시트(프레임 동일 캔버스) + 프레임수·FPS 텍스트. 또는 개별 프레임 PNG(0001…). 권장 ~10fps.
- **납품**: 자산별 개별 PNG + 애니 시트 + **원본(PSD/클립/Aseprite)**. 배경은 레이어 분리본 동봉.
- **라이선스**: 상업 사용·수정·재배포 가능(전권 양도 권장), 서면 명시.
- **파일명**: §6 네이밍 규칙대로(엔진 드롭인용).

## 2. 전체 자산 목록 (현재 구현 전부)

### A. 주인공 (외형 승급 3티어)
환생할수록 외형이 승급한다. 같은 인물, 점점 화려해짐.
| 키 | 설명 |
|---|---|
| `hero_t1` | 기본 — 풋내기 기사, 밝고 단정 |
| `hero_t2` | 승급 — 은빛 정예 기사, 망토·문양 추가 |
| `hero_t3` | 최종 — 금장 영웅, 빛나는 무기·오라 |
- 티어별 애니 4종: **idle**(2~4f), **attack**(3~5f), **hit**(1~2f), **death**(3~5f).
- (정지본만 발주도 가능 — 예산 따라. §5 참고.)

### B. 일반 적 (15종, 3지역 × 5)
지역별 5종. 실루엣 공유 없이 **각자 다른 생물** 권장(예산 시 베이스 공유+팔레트 변주 허용).
- **1지역 갈라진 변경**: 들개 · 부패 정령(녹색 슬라임/정령) · 균열 고블린 · 가시벌레(거미) · 그림자 망령(유령)
- **2지역 화산(잿빛 협곡)**: 잿불 박쥐 · 용암 슬라임 · 그을린 늑대 · 재 정령 · 불씨 골렘
- **3지역 설원(얼어붙은 심연)**: 서리 망령 · 얼음 슬라임 · 동토 늑대 · 심연 술사 · 한기 거미
- 애니(권장): idle(2~4f) + hit(1~2f). (정지본도 가능.)

### C. 보스 (12종 = 전투 스프라이트 + 초상화)
일반 적보다 **훨씬 크고 화려·위협적**. 전투 스프라이트(횡전투용) + 초상화(도전 컷 512²) 각 1.
| 지역 | 보스 | 생물·형태 |
|---|---|---|
| 1 | 거대 거미 아라크 | **거대한 거미** — 다리·붉은 눈, 등에 균열 무늬 |
| 1 | 이끼 골렘 모스 | **돌+이끼 골렘** — 바위 몸체, 초록 이끼 |
| 1 | 망령군주 베일 | **유령형 망령왕** — 후드·해골, 흐릿한 하반신 |
| 1 | 균열의 수호자 | **뿔 갑옷 거인** — 균열 에너지 두른 수호기 |
| 2 | 화염군주 이그 | **불꽃 군주** — 용암 갈라진 몸, 화염 |
| 2 | 용암 거인 칼 | **용암 바위 거인** — 균열로 용암 빛 |
| 2 | 잿빛 드레이크 | **회색 비룡/와이번** — 날개·뿔 |
| 2 | 협곡의 지배자 | **거대 뿔 마수** — 화염 변주 |
| 3 | 빙결군주 헬름 | **얼음 갑옷 군주** — 청록 변주 |
| 3 | 심연 거미 여왕 | **푸른 거대 거미 여왕** — 얼음 가시 |
| 3 | 얼어붙은 망령왕 | **얼음 망령왕** — 서리 영기, 깨진 왕관 |
| 3 | 심연의 지배자 | **심연 대마법사/마왕** — 로브·지팡이, 청보라 마력 |
- 보스 애니(권장): idle(2~4f) + attack(3~5f) + hit + death. (정지본도 가능.)
- **초상화 12종** 각 보스 생물과 일치(512×512, 반신, 화려).

### D. 배경 (3지역)
밝은 야외 톤. **원경/중경/바닥 레이어 분리**(패럴랙스). 캐릭터는 화면 하단 바닥선(세로 960 중 ~620 지점)에 섬.
| 키 | 지역 | 테마(밝게) |
|---|---|---|
| `bg_region1` | 갈라진 변경 | 초원·고대 유적, 푸른 하늘, 멀리 균열 빛 |
| `bg_region2` | 잿빛 협곡 | 화산 협곡, 주황 용암·붉은 노을 |
| `bg_region3` | 얼어붙은 심연 | 설원·빙벽, 청록 오로라 |

### E. UI 키트 (9슬라이스·상태별)
- **패널 프레임** 2종: 일반(`ui_panel`)·강조(`ui_panel_hi`). 9슬라이스(모서리 안전영역 표시). 반투명 어두운 베이스 + 금테.
- **버튼** 3상태(normal/pressed/disabled) × 3변형: 일반(`ui_btn`)·금테 강조(`ui_btn_gold`)·위험 적색(`ui_btn_red`). 9슬라이스.
- **원형 스킬 버튼 프레임** `ui_skill_frame`(강타·가속용, 빈 프레임).
- **바**(테두리+채움 분리): 체력 `ui_bar_hp`, 경험치 `ui_bar_exp`, 진행 `ui_bar_prog`.
- **상단 HUD 프레임**(선택) — 재화/레벨/스테이지 띠.

### F. 아이콘 (15종, 256×256, 금테 프레임 통일)
| 키 | 의미 | 심볼 |
|---|---|---|
| `icon_atk` | 공격력 | 검 |
| `icon_hp` | 체력 | 하트 |
| `icon_def` | 방어력 | 방패 |
| `icon_crit` | 치명타 | 교차 단검/번개 |
| `icon_gold` | 골드(재화) | 금화 더미 |
| `icon_crystal` | 균열석(환생 재화) | 빛나는 크리스탈 |
| `icon_exp` | 경험치 | 별/룬 |
| `icon_level` | 레벨 | 휘장/계급장 |
| `icon_prestige` | 환생 | 소용돌이/회전 화살표 |
| `icon_mission` | 임무 | 두루마리/체크리스트 |

| `icon_offline` | 오프라인 보상 | 모래시계 |

### G. 이펙트 (선택 — 현재 코드 절차 생성, 교체 시 타격감↑)
가로 시트. `fx_slash`(베기 호) · `fx_hit`(타격 스파크) · `fx_crit`(치명 폭발) · `fx_levelup`(상승 빛) · `fx_prestige`(환생 빛기둥) · `fx_boss_appear`(보스 등장 충격파). + 잔불/광원 텍스처는 절차 유지 가능.

### H. 로고·타이틀·앱아이콘
- `logo` — "균열기사" 워드마크(가로, 투명).
- `title_splash` — 첫 실행 타이틀(540×960, 주인공+균열 배경, 밝게).
- `app_icon` — 스토어/웹용 아이콘(1024×1024, 주인공 얼굴/엠블럼).

## 3. 화면 내 사용 맥락 (스케일·프레이밍 참고)

- **전투 화면(메인)**: 좌측 주인공(오른쪽 봄) ↔ 우측 적/보스(왼쪽 봄), 하단 바닥선. 주인공 키 ~화면의 1/7, 보스 ~1/4(주인공 1.5~2배).
- **보스 도전 컷**: 보스 초상화가 화면 중앙에 크게(≈340px) 떠오름 + 이름. → 초상화는 **반신·정면·강렬**하게.
- **상단 HUD**: 좌 골드(코인 아이콘)·우 레벨/전투력, EXP 바.
- **하단 성장 패널**: 5행(공격력/체력/방어력/치명타/골드획득), 각 행 좌측 아이콘.
- **오버레이**: 환생·균열석 상점 / 임무 / 업적 / 설정 — 패널+버튼+아이콘 조합.

- UI 키트 1세트 (패널 2 + 버튼 3변형×3상태 + 스킬프레임 + 바 3)
- 아이콘 15종
- 이펙트 6종 (선택)
- 로고 + 타이틀 + 앱아이콘

## 5. 우선순위 / 예산 3단계

**💧 Lite (즉시 체감, 정지본 위주)**: 주인공 3티어(정지) + 적 15(정지) + 보스 12 정지 + 초상화 12 + 배경 3 + UI 키트 + 아이콘 15 + 로고.
**🔷 Standard (출시 퀄리티)**: Lite + 주인공·보스 **애니 4종**, 적 idle/hit 애니, 이펙트 6, 타이틀/앱아이콘.
**💠 Premium**: Standard + 4지역 이상 확장 세트, 추가 외형 티어, 펫/탈것, UI 모션 에셋.

> **테스트 선납품 필수**: 주인공 t1 + 보스 2종(서로 다른 생물, 예: 거미+골렘) + 배경 1 + 적 2 먼저 받아 **밝기·SD비율·대비·생물일치** 확인 후 본작업.
