# HANDOFF

## 현재 목표

강타·연격·반격이 하나의 강한 적(오우거 징수꾼 엘리트)을 상대로 서로 다른 공략으로 느껴지는지 확인한다. (TASK_008)

- 아이폰 테스트용 웹(HTML5) 내보내기 + Tailscale HTTPS 노출, 한글 폰트 적용(폰트 비공개).

- TASK_003: 경험치 +5, 레벨/경험치 바, 레벨업·연출. 아이폰 확인됨.
- TASK_004: 체력·방어력 강화, 방어력 피해 계산(최소 1), 적 공격력 4→6 상쇄, 강화 버튼 3개, 피해 숫자. 아이폰 확인됨(선택 재미 검증됨).
- TASK_005: 레벨 10마다 특성 포인트, 특성 패널(투자·무료 초기화), 강타(4타째 220%)·연격(30% 60% 추가 베기)·반격(25% 40%↓ + 120%), 충돌 우선순위, 특성별 전투 문구. 아이폰 확인됨(대만족). 템포 단축·연격/반격 연출 강조.
- TASK_006: 일반 적 3종(늑대 hp32/atk5·고블린 hp22/atk4·방패병 hp75/atk8/def5)과 결정적 순환(늑대3→고블린3→방패병1), 크기·실루엣·이름·무리 번호 구분, 적별 보상(5/5·4/4·12/10), 방패병에 강타 유리. 검증 모드는 기준 적만 생성해 기존 검증 보존.
- TASK_007: 반격을 결정적(적 3번째 공격마다 확정 막기 40%↓ + 120% 반격)으로 바꿔 일반 적에게도 보이게. 고블린은 3마리 동시 줄지어 등장(겹침 없이)·앞에서부터 한 마리씩 전투·죽으면 전진(단일 대상 유지). 아이폰 확인됨(만족).
- TASK_008: 일반 순환 뒤 오우거 징수꾼 엘리트(체력 240·공16·방3·보상 40/30). 공격 패턴 기본·기본·강공격(약 1초 준비 동작, 준비 중 무피해·사망 시 취소). 강공격 round(16×2.0)−방어. 반격이 3번째 강공격에 맞춰 발동(강공격 30→18). 큰 몸체·몽둥이·[엘리트] 이름표. 처치 시 골드+40·경험치+30 1회. 패배 시 첫 늑대 복귀→일반 순환 후 재도전(성장 유지, 오우거 체력 240 초기화). UNKNOWN.md·HANDOFF 반격 규칙 정본 동기화.

## 특성 작동 요약

- 강타: 기본 공격 4번째마다 round(공격력×2.2)−방어력 피해. 큰 전진·적 밀림·큰 금색 문구.
- 연격: 일반 기본 공격 시 30% 확률로 round(공격력×0.6)−방어력 추가 베기 1회(재발동 없음, 강타 공격엔 미발동).
- 반격: 적의 3번째 공격마다 받는 피해를 40% 줄이고(×0.6), 생존 시 round(공격력×1.2)−방어력로 즉시 반격.
- 일반 플레이는 확률, 헤드리스 검증은 force 플래그 강제 경로.

## 실행 방법

방법 A — 에디터로 실행 (권장, 화면 확인용).

```
"C:\Users\USER\godot-engine\Godot_v4.7-stable_win64.exe" --path "C:\Users\USER\unknown"
```

에디터가 열리면 F5(실행)로 전투 화면을 본다. 또는 다음으로 바로 실행한다.

```
"C:\Users\USER\godot-engine\Godot_v4.7-stable_win64.exe" --path "C:\Users\USER\unknown" Battle.tscn
```

방법 B — 자동 검증 (화면 없이 로그만).

```
"C:\Users\USER\godot-engine\Godot_v4.7-stable_win64_console.exe" --headless --path "C:\Users\USER\unknown" -- --verify
```

## 검증 상태

- 헤드리스 자동 검증(--verify): 기존 TASK_001~007 + TASK_008 오우거 프로필·순서·공격 패턴(기본·기본·강공격)·피해(기본14·강공격30)·반격 연동(강공격 30→18, 3번째만)·보상(40/30 1회)·패배 후 복귀 **ALL PASS TASK_001~008**, 종료 코드 0, 경고/오류 0건. 무작위 비의존.
- 라이브(비검증) 오우거 도달·전투·패배 복귀 55초 스모크: 런타임 오류 0.
- PC 화면 검증: 미실행(헤드리스만). 아이폰 검증: 대기.

## 남은 위험

- 오우거가 엘리트로 느껴지는지, 강공격 준비 동작 가독성, 반격이 강공격 막는 역할로 분명한지, 강타·연격의 공략 차이, 전투 길이·재도전 간격은 아이폰에서 직접 확인 필요(미검증).
- 강공격 준비 동작·몽둥이 들기 연출은 헤드리스라 로직만 확인됨(화면 동작 미확인). 어색하면 조정 가능.
- 무특성·저성장 상태로는 오우거에게 패배하기 쉬움(설계된 재도전 루프). 전투 길이/난이도가 답답하면 체력·간격·준비시간 조정.
- 사망/재시작 연출이 약한 점은 그대로 보류.
