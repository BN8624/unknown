- TASK_007: 반격을 결정적(적 3번째 공격마다 확정 막기 40%↓ + 120% 반격)으로 바꿔 일반 적에게도 보이게. 고블린은 3마리 동시 줄지어 등장(겹침 없이)·앞에서부터 한 마리씩 전투·죽으면 전진(단일 대상 유지). 강타·연격 수치 불변.

## 특성 작동 요약

- 강타: 기본 공격 4번째마다 round(공격력×2.2)−방어력 피해. 큰 전진·적 밀림·큰 금색 문구.
- 연격: 일반 기본 공격 시 30% 확률로 round(공격력×0.6)−방어력 추가 베기 1회(재발동 없음, 강타 공격엔 미발동).
- 반격: 피격 시 25% 확률로 받는 피해 ×0.6(40% 감소), 생존 시 round(공격력×1.2)−방어력로 즉시 반격.
- 일반 플레이는 확률, 헤드리스 검증은 force 플래그 강제 경로.

방법 A — 에디터로 실행 (권장, 화면 확인용).

```

```

에디터가 열리면 F5(실행)로 전투 화면을 본다. 또는 다음으로 바로 실행한다.


## 검증 상태

- 헤드리스 자동 검증(--verify): 반격 결정적(발동열 F,F,T,F,F,T) + 기존 TASK_001~006 **ALL PASS TASK_001~007**, 종료 코드 0, 경고/오류 0건. 무작위 비의존.
- 라이브(비검증) 늑대→고블린 무리→방패병 26초 스모크: 런타임 오류 0.
- PC 화면 검증: 미실행(헤드리스만). 아이폰 검증: 대기.

## 남은 위험

- 반격이 일반 적에게도 또렷이 보이는지, 세 특성이 확실히 다르게 느껴지는지, 고블린 3마리 줄세우기가 무리로 보이는지는 아이폰에서 직접 확인 필요(미검증).
- 고블린 무리 줄·전진 연출은 헤드리스라 로직만 확인됨(화면 동작 미확인). 줄 간격(82px)·전진감이 어색하면 조정 가능.
- 성장 속도: 방패병(12/10)이 섞여 약간 빨라질 수 있음. 급변하면 후속 조정.
- 사망/재시작 연출이 약한 점은 그대로 보류.
