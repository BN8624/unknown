# TASK 005 — 레벨 10 전투 특성 선택 루프 구현

## 작업 목적

레벨 10마다 전투 특성 포인트를 지급하고, 강타·연격·반격 중 하나에 투자해 실제 전투 행동을 바꾼다.

```text
자동 전투와 성장
→ 레벨 10 도달
→ 전투 특성 포인트 1 획득
→ 강타·연격·반격 중 하나 선택
→ 공격 또는 방어 행동이 실제로 변화
→ 무료 초기화로 다른 특성과 비교
```

핵심 질문.

> 강타·연격·반격이 단순한 능력치 증가가 아니라 서로 다른 플레이 스타일로 보이고 느껴지는가.

특성 선택과 효과를 한 작업에서 구현한다. 포인트만 주고 효과를 미루지 않는다.

## 유지할 기능

자동 전진, 적 등장·접근, 자동 공격, 처치 반복, 골드·경험치, 공격력·체력·방어력 강화(독립 비용), 레벨업(+공격 1·+체력 5·완전 회복), 방어력 피해 계산(최소 1), 피해 숫자, 사망 재시작, TASK 001~004 검증, Web·한글 폰트. 기존 수치는 변경하지 않는다.

## 구현 범위

### 1. 특성 포인트 상태

`trait_points`, `power_trait_level`(강타), `combo_trait_level`(연격), `counter_trait_level`(반격), `total_trait_points_earned`. 각 계열 최대 1단계(0 또는 1). 2·3단계는 구현하지 않는다.

### 2. 포인트 지급

10레벨 단위 도달마다 +1. 여러 구간을 한 번에 넘어도 누락 없이, 같은 레벨에서 중복 없이 지급. `지급 총합 = 현재 레벨 ÷ 10(정수)`. 레벨 10의 첫 포인트가 주요 검증 대상.

### 3. 레벨 10 알림

처음 레벨 10 도달 시 기존 레벨업 연출과 별도로 "전투 특성 포인트 획득! 강타·연격·반격 중 하나를 선택하세요." 약 1~1.5초 표시 후 특성 화면 자동 열기. 닫아도 포인트 유지, 미사용 닫기 허용.

### 4. 특성 화면

전투 화면 위에 뜨는 간단한 패널(새 씬 금지). 표시: 보유 포인트, 강타 0/1, 연격 0/1, 반격 0/1, 각 효과 설명, 무료 초기화, 닫기. 모바일에서 글자 잘림·겹침 없음.

- 강타: 4번째 기본 공격마다 220% 피해의 내려찍기.
- 연격: 기본 공격 시 30% 확률로 60% 피해의 추가 베기.
- 반격: 피격 시 25% 확률로 피해 40% 감소 후 120% 피해 반격.

### 5. 투자 규칙

포인트 1 이상일 때만 투자. 투자 시 해당 단계 +1, 포인트 −1, UI·메인 표시 갱신. 각 계열 최대 1단계, 이미 1이면 추가 불가, 포인트 0이면 버튼 비활성.

### 6. 무료 초기화

강타·연격·반격을 0으로, 투자 포인트 전부 보유로 반환, 특성 전투 카운터 초기화, UI 갱신. 골드·경험치·레벨·기본 강화는 유지. 비용·확인 팝업 없음.

## 전투 특성 효과

### 7. 강타 1단계

기본 공격 횟수를 세어 4번째 공격을 강타로. 1·2·3 일반, 4 강타, 이후 다시 카운트. 적이 바뀌어도 유지, 사망·초기화 시 0. 반격·연격 추가 공격은 카운트에 포함하지 않음.

```text
기술 공격력 = 반올림(현재 공격력 × 2.2)
최종 피해 = 기술 공격력 − 적 방어력, 최소 1
```

보스 추가 피해는 보류. 표현: 긴 준비 동작, 큰 내려찍기 전진, 큰 글자 + "강타" 문구(예 `강타 -22`), 가능하면 적 짧게 밀림. 충격파·화면 흔들림 없음.

### 8. 연격 1단계

기본 공격 적중 후 한 번 판정, 발동 시 짧은 간격 뒤 추가 베기 1회(일반 대기시간 없음). 추가 베기는 연격 재발동 안 함. 반격 공격에서 발동 안 함. 강타가 발동한 공격에서는 판정 안 함. 첫 공격으로 적이 죽으면 추가 베기 없음.

```text
기술 공격력 = 반올림(현재 공격력 × 0.6)
최종 피해 = 기술 공격력 − 적 방어력, 최소 1
```

확률 판정은 한 곳에서. 일반 30%. 검증은 강제 발동/미발동 경로 사용. 표현: 짧은 추가 전진, 잔상/선, "추가 베기" 문구(예 `추가 베기 -6`).

### 9. 반격 1단계

```text
적 공격 → 반격 발동 판정 → 발동 시 받는 피해 40% 감소 → 적용 → 생존 시 즉시 반격
```

감소 피해로 사망하면 반격하지 않음.

```text
기본 실제 피해 = 적 공격력 − 용병 방어력, 최소 1
반격 발동 피해 = 반올림(기본 실제 피해 × 0.6), 최소 1   (예: 4 → 2)
반격 피해 = 반올림(현재 공격력 × 1.2) − 적 방어력, 최소 1
```

반격 공격은 강타 카운트에 미포함, 연격 미발동. 일반 25%, 검증은 강제 경로. 표현: 방어 표시, 감소 피해 숫자, 즉시 반격 동작, "방어"/"반격" 문구(예 `방어 -2`, `반격 -12`).

---

## 특성 충돌 우선순위

기본 공격: 1) 강타 발동 확인 → 2) 아니면 일반 공격 → 3) 일반일 때만 연격 판정. 강타 공격에서 연격 발동 안 함. 반격은 적 공격에 대한 별도 반응으로, 반격 공격에서는 강타·연격 발동 안 함. 별도 시너지 없음.

## 자동 검증

기존 TASK 001~004 검증을 삭제하지 않고 계속 통과. 추가로 다음을 결정적으로(확률 비의존, 강제 발동 경로) 검증.

- 레벨 10 지급: 레벨 9 총 0, 레벨 10 총 1·보유 1, 같은 레벨 중복 없음(지급 함수 재호출해도 추가 없음).
- 투자/초기화: 보유 1 → 강타 투자 → 강타 1·보유 0, 포인트 0이면 타 특성 투자 불가. 초기화 → 강타·연격·반격 0·보유 1, 레벨·골드·경험치·기본 강화 유지, 강타 카운터 0.
- 강타: 1·2·3 일반, 4 강타, 다음 다시 1. 공식 `반올림(atk×2.2) − def`, 최소 1. 반격·연격이 강타 카운터를 증가시키지 않음.
- 연격: 강제 발동 시 기본 1회 + 추가 베기 1회, 추가 베기 재발동 없음. 강제 미발동 시 추가 없음. 첫 공격으로 죽으면 추가 없음. 공식 `반올림(atk×0.6) − def`.
- 반격: 강제 발동 시 받는 피해 `반올림(4×0.6)=2`, 반격 `반올림(atk×1.2) − def`. 미발동 시 피해 4·반격 없음. 감소 피해로 사망 시 반격 없음.
- 충돌: 강타+연격 동시, 4번째는 강타만·연격 미발동. 반격 공격에서 강타 카운터 미증가·연격 미발동.
- 능력치 누적 불변식 유지.

종료 조건: 위 + TASK 001~004 모두 통과 시 종료 0. 확률 결과로 성공/실패가 갈리면 안 됨. 실패 시 예상/실제 로그 후 종료 1. 무한 실행 방지 제한 유지.

## 재미 확인 질문

1. 강타·연격·반격이 다른 플레이 스타일로 느껴지는가. 2. 선택 특성 때문에 더 지켜보고 싶은가. 3. 초기화 후 다른 특성도 시험하고 싶은가. 4. 특정 특성이 무조건 정답은 아닌가. 5. 적 하나로도 특성 차이가 보이는가. 차이가 약하면 적을 늘리지 말고 발동 빈도·동작·배율·문구·연출·속도를 먼저 수정.
