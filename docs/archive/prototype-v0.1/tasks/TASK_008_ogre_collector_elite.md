# TASK 008 — 오우거 징수꾼 엘리트와 재도전 루프 구현

## 작업 목적

일반 적 순환 뒤에 첫 엘리트 오우거 징수꾼을 추가한다. 높은 체력과 느리지만 강한 공격(준비 동작 있는 강공격)을 사용한다.

```text
일반 적 사냥 → 오우거 등장 → 기본·기본·강공격 → 특성별 다른 대응
→ 승리 시 큰 보상 → 패배 시 일반 사냥 복귀 → 성장 후 재도전
```

핵심 질문: 강타·연격·반격이 하나의 강한 적을 상대로 서로 다른 공략으로 느껴지는가.

엘리트 한 종류만. 철퇴의 브루노 보스·빚·명성은 구현하지 않는다.

---

`CLAUDE.md`, `UNKNOWN.md`, `HANDOFF.md`, `TASK_005`~`TASK_007`, `checklist.md`, `context-notes.md`, 현재 `Battle.gd`. 기존 TASK 001~007 기능·검증 유지, 전투/성장 구조 전면 재작성 금지.

## 정본 문서 동기화 (TASK 007 승인 반영)

- `UNKNOWN.md` 반격 1단계: "피격 시 25% 확률로 …" → **"적의 3번째 공격마다 피해를 40% 줄이고 즉시 반격한다."** 나머지 수치(40% 감소, 120% 반격, 방어 성공 표시) 유지.
- `HANDOFF.md` 특성 요약의 반격 설명도 "적의 3번째 공격마다 …"로 수정.
- 기획 변경이 아니라 TASK 007에서 승인·확인된 규칙의 정본 동기화. `context-notes.md`의 TASK 007 결정은 유지.

## 오우거 징수꾼 프로필

이름 오우거 징수꾼, 종류 엘리트. 체력 240, 공격력 16, 방어력 3, 경험치 30, 골드 40, 기본 공격 간격 2.0초, 접근 속도 90~110px/s(느림). 프로필 원본은 복사해 사용, 별도 데이터 파일·편집기 없음.

---

## 등장 순서 / 재도전

일반 순환(늑대3→고블린3→방패병1) 뒤 오우거. 전체: 늑대3→고블린3→방패병→오우거→(처치 시) 다시 늑대. 무작위 없음.

오우거에게 패배(용병 사망) 시 즉시 재등장하지 않고: 첫 늑대로 복귀 → 일반 한 순환 동안 성장 → 방패병 처치 후 오우거 재도전. 패배해도 레벨·경험치·골드·강화·특성 유지. 재도전 시 오우거 체력 240으로 초기화. 패배 중에는 골드·경험치 미지급.

---

## 등장 표현 / 외형

등장 시 중앙(또는 전투 영역 상단)에 "엘리트 등장! 오우거 징수꾼" 0.8~1.2초. 장시간 정지 금지, 짧은 등장 정지 허용. 외형(임시 도형): 일반 적보다 큰 몸체·넓고 무거운 실루엣, 갈색/주황 계열, 손에 큰 몽둥이 도형, 넓은 체력 바, 이름표 "[엘리트] 오우거 징수꾼". 색만 다른 방패병처럼 보이면 안 됨(크기+몽둥이 함께 달라야).

---

## 공격 패턴

세 공격이 한 주기: 1 기본 → 2 기본 → 3 강한 몽둥이 내려찍기, 반복. 사망·전투 취소 시 패턴 카운터 초기화.

- 기본 공격: 공격력 16. 실제 피해 = 16 − 용병 방어력(2 → 14), 최소 1. 기존 피해 숫자 사용.
- 강공격: 기술 공격력 = round(16 × 2.0)=32. 실제 피해 = 32 − 방어력(2 → 30), 최소 1.
- 강공격 전 약 1초 준비 동작: 몽둥이 들어올림, 몸체 주황 강조, "강공격 준비!" 문구, 준비 중 피해 없음. 준비 중에도 용병은 계속 공격. 준비 중 오우거가 죽으면 강공격 취소. 죽은 용병/오우거가 공격 이어가지 않음.

---

## 특성 연동 (수치 변경 없음)

- 반격: 오우거의 3번째 공격(강공격)에 맞춰 확정 발동 → 강공격 피해 40% 감소(round(30×0.6)=18) → 생존 시 round(공격력×1.2)−3 반격. 강공격 방어를 일반보다 분명히 표시(예 "강공격 방어 -18", "반격 -N"). 같은 강공격에 반격 두 번 없음. 1·2 기본엔 반격 없음. 반격 공격은 강타 카운터·연격 발동 안 함.
- 강타: round(공격력×2.2)−3. 오우거 전용 추가 피해·관통 없음. 높은 체력을 큰 단위로 줄이는 모습만.
- 연격: 기존 규칙(30% 추가 베기 60%, 강타 공격엔 미발동, 재발동 없음). 느린 공격·준비 사이에 추가 타격으로 유리. 전용 중첩·자세 붕괴 없음.

특성별 목표: 강타=큰 피해로 빠르게, 연격=느린 공격 사이 많은 타격, 반격=예고된 강공격 막고 되받기. 어떤 특성으로도 성장 충분하면 처치 가능. 특정 특성 없으면 절대 못 이기는 구조 금지.

---

## 보상 / 상태

처치 시 골드 +40, 경험치 +30을 한 번만(반격 처치도 1회). 즉시 레벨업 조건 넘으면 레벨업. 짧은 결과 문구 "엘리트 처치! 골드 +40 경험치 +30". 빚·명성 없음.

오우거 상태는 현재 enemy Dictionary에 최소만 추가(is_elite, attack_count, is_winding_up, windup_timer 등). 범용 보스 AI·복잡한 상태머신 없이 기존 WALK/COMBAT/DEAD 안에서 최소 처리.

오우거는 고블린 무리와 동시 존재 금지(등장 전 고블린 대기열·이전 노드 정리 확인). 오우거 사망·용병 사망 시 몸체·체력 바·이름표·몽둥이·준비 문구·강조 효과 모두 제거. 이전 엘리트 UI가 다음 늑대 전투에 남지 않게.

임시 시작 특성 포인트(DEBUG_START_TRAIT_POINTS=3)는 유지(검증 무영향, 정식 빌드 전 제거 주석 유지). 이번 작업에서 제거하지 않음.

## 자동 검증

기존 TASK 001~007 검증 모두 유지(삭제·완화 금지). 추가:

- 프로필: 체력 240, 공격력 16, 방어력 3, 경험치 30, 골드 40, 간격 2.0.
- 순서: 늑대3·고블린3·방패병·오우거·늑대. 오우거 처치 후 첫 늑대.
- 공격 패턴: 첫 6회 기본·기본·강공격·기본·기본·강공격. 강공격이 준비 완료 전 피해 주면 실패. 준비 중 사망 시 강공격 취소.
- 피해: 기본 14, 강공격 30(round(16×2.0)−2), 최소 1.
- 반격 연동: 3번째=강공격, 확정 발동, 강공격 30→감소 18, 생존 시 반격 1회. 같은 강공격에 반격 두 번 없음, 1·2 기본엔 미발동.
- 보상: 골드+40, 경험치+30, 1회. 반격 처치도 1회. 오우거가 용병 처치 시 보상 없음.
- 패배 후 복귀: 오우거·UI 제거, 부활, 다음 적 첫 늑대, 일반 순환 후 오우거 재등장, 성장 유지, 오우거 체력 240 초기화.

종료: 위 + 기존 모두 통과 시 종료 0(ALL PASS TASK_001~008). 실패 시 예상/실제 로그 후 종료 1. 검증 모드는 기준 적만 생성(기존 유지), 오우거는 결정적·라이브 스모크로 확인.

## 완료 조건

UNKNOWN.md·HANDOFF.md 반격 동기화. 일반 순환 후 오우거 등장·구분. 기본 2번 후 강공격, 약 1초 준비(준비 중 무피해), 준비 중 사망 시 취소. 반격이 3번째 강공격에 발동. 강타가 큰 단위 감소. 연격이 느린 공격 사이 추가 타격. 어떤 특성으로도 처치 가능. 처치 시 골드 40·경험치 30 한 번. 패배 시 첫 늑대 복귀, 다음 순환 후 재도전, 성장 유지. 고블린·오우거 노드 겹침 없음. 기존 TASK 001~007 검증 유지, TASK 008 포함 종료 0, 오류·경고 없음, 범위 밖 없음.

## 검증 / 재미 판단

라이브 스모크로 최소 한 번의 오우거 전투(방패병 다음 등장, 등장 문구, 큰 몸체, 기본 2번 후 준비 동작, 반격 막기, 강타 큰 감소, 연격 추가 타격, 패배·처치 후 첫 늑대 복귀, 오류 없음). 못 한 항목은 미완료 기록.

아이폰 사용자: 엘리트로 즉시 느껴지는지, 강공격 예고 인지, 반격이 강공격 막는 역할로 분명한지, 강타가 빠르게 줄이는지, 연격이 몰아넣는지, 특성 하나만 정답 아닌지, 패배 후 다시 잡고 싶은지, 재등장 간격·전투 길이·보상 체감 적당한지. 사용자 확인 전 완료로 기록 안 함.

오우거가 체력만 많은 방패병처럼 느껴지면 다음으로 가지 말고 준비 동작·강공격 주기·공격 간격·반격 표현·체력·크기·등장 연출을 먼저 조정(특성 수치는 우선 불변).
