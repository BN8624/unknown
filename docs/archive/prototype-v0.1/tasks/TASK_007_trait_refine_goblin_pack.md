# TASK 007 — 특성 차별화 정비와 고블린 무리 표현

## 작업 목적

방치형(완전 자동) 정체성을 유지하면서 세 특성의 차별점을 분명히 하고, 고블린을 "무리"로 보이게 한다.

TASK 005·006 플레이 피드백.

- 적 구분은 됨.
- 고블린이 무리로 안 느껴짐.
- 강타·연격은 잘 박히나 반격이 약하고, 세 특성 차별점이 부족함.

사용자 결정(2026-06-24): 방치형 유지, 특성을 역할·연출·리듬으로 차별화. 반격을 확률 → 결정적으로 변경(`UNKNOWN.md`의 25% 확률과 달라지는 기획 변경, 사용자 승인됨). 고블린은 3마리 동시 등장(겹침 없이 줄지어)하되 전투는 앞에서부터 한 마리씩.

---

## 먼저 읽을 문서

`CLAUDE.md`, `UNKNOWN.md`, `HANDOFF.md`, `TASK_001`~`TASK_006`, `checklist.md`, `context-notes.md`. `UNKNOWN.md`·`CLAUDE.md`는 수정하지 않는다(반격 변경은 context-notes에 기록).

---

## 구현 범위

### 1. 반격 재설계 — 확률 → 결정적 리듬

- 변경 전: 피격 시 25% 확률.
- 변경 후: 반격 1단계일 때 **적의 3번째 공격마다** 확정으로 막고 반격한다.
  - 받는 피해 = round(기본 실제 피해 × 0.6), 최소 1. (예: 4 → 2)
  - 반격 피해 = round(공격력 × 1.2) − 적 방어력, 최소 1.
- 강타가 "내 4타째 큰 한방"이면 반격은 "적 3타째 막고 되받기"로 대칭. 일반 적에게도 규칙적으로 보이고, 느린 강공격 적(다음 엘리트)에서 진가.
- 받는 공격 카운트는 반격 1단계일 때만 센다. 용병 사망·특성 초기화 시 0으로.
- 강타·연격 수치·동작은 변경하지 않는다(만족도 높음). 연격 확률은 그대로 30%.
- 막은 피해로도 사망하면 반격하지 않는다. 반격 공격에 강타 카운트 증가·연격 발동 없음(기존 규칙 유지).

### 2. 고블린 3마리 동시 등장(줄지어, 안 겹침)

- 고블린 무리 차례가 되면 3마리를 동시에 가로로 줄지어 등장시킨다(겹치지 않는 간격).
- 전투는 단일 대상 유지: 가장 앞 고블린만 용병과 싸운다. 동시 다중 전투·멀티타겟·범위 공격은 만들지 않는다.
- 앞 고블린이 죽으면 뒤 고블린들이 앞으로 전진해 다음이 전투한다(거의 끊김 없이).
- 무리 전체(3마리)를 처치하면 일반 지연 후 방패병으로.
- 용병 사망으로 무리가 중단되면 현재 무리를 처음부터 다시 시작한다(순환 위치 유지).
- 적 교체·무리 종료 시 모든 고블린의 도형·체력 바·이름표가 남지 않게 정리한다.

### 3. 유지

자동 전진·전투·순환(늑대3→고블린3→방패병1), 골드·경험치·강화·레벨업·완전 회복, 강타·연격, 적 프로필 수치·보상, 템포, 피해 숫자, Web·한글 폰트, TASK 001~006 검증. 적 프로필 수치는 변경하지 않는다.

## 자동 검증

기존 TASK 001~006 검증을 유지하되, 반격 검증을 결정적 규칙으로 갱신한다.

- 반격: 반격 1단계에서 받는 공격 카운트가 1,2,3,... 진행할 때 **3번째마다만** 발동(예: 발동열 [F,F,T,F,F,T]). 받는 피해 round(4×0.6)=2, 반격 피해 round(공격력×1.2)−방어력. 초기화·사망 시 카운트 0.
- 강타·연격·충돌·지급·투자/초기화·프로필·순서·보상·전투 역할·능력치 불변식은 기존대로 통과.
- 확률 결과로 성공/실패가 갈리지 않게(연격만 강제 경로 유지, 반격은 결정적이라 force 불필요).
- 고블린 줄세우기는 런타임 무오류로 확인(헤드리스 라이브 스모크). 검증 모드는 기준 적만 생성(기존 유지).

종료: 모두 통과 시 종료 0(ALL PASS TASK_001~007 표기). 실패 시 예상/실제 로그 후 종료 1.

## 구현하지 말아야 할 것

동시 다중 전투·멀티타겟·범위 공격, 적이 실제로 쌓여 동시에 때리는 시스템, 특성 2·3단계, 액티브 스킬 전환, 프리셋, 엘리트·보스, 빚·명성, 저장, 사운드, 완성 아트, 대규모 리팩터링. 단일 Battle.gd에서 관리 가능하면 구조 유지.

## 완료 조건

반격이 적 3번째 공격마다 확정 발동·막기·되받기, 일반 적에게도 규칙적으로 보임. 강타·연격 불변. 고블린 3마리 동시에 줄지어 등장(안 겹침), 앞에서부터 한 마리씩 전투, 죽으면 전진. 무리 후 방패병. 사망 시 무리 처음부터. 적 교체 시 잔존 없음. 기존 성장·특성·순환 정상. TASK 001~006 검증 유지 + 반격 결정적 검증 통과. 종료 0, 오류·경고 없음, 범위 밖 없음.
