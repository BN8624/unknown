# 균열기사 (Riftblade Knight)

세로형(540×960) 모바일 **방치형 RPG**. Godot 4.7로 만든 작은 출시 후보 v0.1.
갈라진 변경의 균열에서 끝없이 밀려오는 마물을 홀로 베어내는 기사를 자동 전투로 성장시킨다.

> 정본 기획: [`PRODUCT_RESET_SPEC.md`](PRODUCT_RESET_SPEC.md)

---

## 현재 상태

**출시 후보 v0.1 — 핵심 루프와 메타·콘텐츠가 닫힌 상태.** 그래픽은 Kenney CC0 픽셀 스프라이트 + 절차적 합성 배경/이펙트, 사운드는 절차적 합성(CC0). 모두 상업 배포 가능.

구현됨:

- 자동 전투(공격·치명타·피해 숫자·**베기 섬광**·파티클), 패배 시 자동 재정비
- 성장 5종: 공격력 · 체력 · 방어력 · 치명타 · 골드 획득(비용 증가·부족 시 비활성)
- 레벨·경험치, 전투력 표시, 큰 수 축약 표기(K/M/B/…)
- 스테이지 진행, 5층마다 보스(스프라이트 확대·오라·HP바·강공격 예열·**분노 페이즈**로 구분)
- **액티브 스킬 2종**: 강타(쿨9s, 8배 타격)·가속(쿨18s, 공속 2배). 자동전투에 탭 요소
- 3개 지역: 갈라진 변경(1~20) · 잿빛 협곡(21~40) · 얼어붙은 심연(41~60), 이후 무한 스케일. 지역별 던전 배경·적
- 보상 표시, 지역 클리어 → 다음 지역 개방
- **환생(프레스티지)** + **균열석 상점**(영구 강화 4종) + **환생 시 영웅 외형 승급**
- **리텐션**: 일일 보상(연속일 보너스) + 회전 임무 3종(처치/전진/강화/보스/골드)
- **오프라인 보상**(최대 8시간)
- 밸런스: 시뮬레이션 검증(`tools/sim.py`) — 첫보스 ~1분·20층 ~5분, 이후 점증 → 환생 유도
- **그래픽**: Kenney 픽셀 캐릭터/몬스터, 횃불 켜진 벽돌 던전 배경, 잔불·비네트 분위기
- **사운드**: 합성 효과음 8종 + BGM 루프, 설정 on/off. 웹/iOS는 자동재생 차단 때문에 **첫 화면 터치 후** 재생 시작
- 저장/불러오기(JSON, 재접속 진행 유지), 설정·진행 초기화
- 첫 실행 온보딩 안내

아직 없는 것(후속 후보): 캐릭터 애니메이션(현재 정지 스프라이트), 더 긴 BGM, 3지역 이상, 적 스킬 다양화, 랭킹/업적.

## 실행

에디터로 실행(화면 확인):

```
"<godot>" --path . Main.tscn
```

Web 빌드:

```
"<godot>" --headless --path . --export-release "Web" "build/index.html"
```

> Web 페이지 `<title>`은 export 시 프로젝트명("unknown")으로 리셋되므로, 내보낸 뒤 `build/index.html`의 title을 "균열기사"로 바꾼다.

개발용 스크린샷 모드: `"<godot>" --path . -- --shot` → `scratchpad/shot_*.png` 저장 후 종료.

## 구조

데이터(수치·콘텐츠)와 화면(루프·UI)을 분리해 나중에 쉽게 바꾸도록 했다.

| 파일 | 역할 |
| --- | --- |
| `Main.gd` / `Main.tscn` | 화면·전투 루프·성장·스테이지·보스·보상·저장·오프라인·환생·사운드 |
| `reset/GameData.gd` | 밸런스·콘텐츠 데이터(적·스테이지·지역·성장·환생 수치) |
| `reset/SaveSystem.gd` | JSON 저장/불러오기(버전 필드) |
| `reset/gen_art.py` | 절차적 텍스처(글로우·잔불·비네트 등) → `assets/gen/` |
| `reset/gen_audio.py` | 효과음·BGM 합성 → `assets/gen_sfx/`, `assets/gen_bgm/` |
| `assets/sprites/` | Kenney Tiny Dungeon에서 추출한 캐릭터·몬스터 픽셀 타일(CC0) |
| `assets/gen/bg_dungeon*.png` | Kenney 타일로 합성한 지역별 던전 배경 |

저장 위치: `user://riftblade_save_v1.json` (레벨·경험치·골드·강화·스테이지·균열석·상점·설정·사운드).

절차적 텍스처/사운드를 다시 만들려면: `python reset/gen_art.py`, `python reset/gen_audio.py` (numpy·Pillow 필요).

## 크레딧 / 라이선스

- **캐릭터·몬스터 스프라이트, 던전 배경 타일**: [Kenney](https://kenney.nl) — *Tiny Dungeon* 팩, **CC0**. `assets/sprites/`, `assets/gen/bg_dungeon*.png`(타일 합성).
- **분위기 텍스처(글로우·잔불·비네트)·효과음·BGM**: `reset/gen_art.py`·`reset/gen_audio.py`로 생성한 **절차적 자체제작(CC0)**. `assets/gen*`.
- AI 이미지 생성 미사용. 모두 CC0라 상업 배포 가능(저작자 표시 의무 없음, 위는 감사 표기).
- 한글 폰트(`malgun.ttf`)는 저장소에 포함하지 않으며(상용), 없으면 기본 폰트로 동작한다.
